This desktop application helps law firms manage clients, matters, billing,
court schedules, and corporate secretary functions with full Arabic/English
bilingual support.

================================================================================
SYSTEM REQUIREMENTS
================================================================================

- Windows 10 or later (64-bit)
- 4 GB RAM minimum (8 GB recommended)
- 500 MB free disk space
- Screen resolution: 1366x768 or higher

================================================================================
INSTALLATION INSTRUCTIONS
================================================================================

1. Double-click "Qanuni Setup 1.0.0.exe"

2. Follow the installation wizard:
   - Choose installation directory (default: C:\Program Files\Qanuni)
   - Select "Create desktop shortcut" (recommended)
   - Click "Install"

3. Launch Qanuni from desktop shortcut or Start Menu

4. On first run, you will see the License Activation screen

================================================================================
LICENSE ACTIVATION
================================================================================

Qanuni requires a valid license key to operate.

STEP 1: Obtain Your License Key
   - Your license key was provided in your purchase confirmation email
   - Format: XXXX-XXXX-XXXX-XXXX-XXXX-XXXX-XXXX-XXXX
   - Each license is bound to one computer (non-transferable)

STEP 2: Get Your Machine ID
   - The License Activation screen displays your unique Machine ID
   - Copy this ID using the "Copy Machine ID" button
   - Send it to support if you need a license key generated

STEP 3: Activate
   - Paste your license key into the activation field
   - Enter your firm name (e.g., "Kallas & Associates Law Firm")
   - Click "Activate License"

STEP 4: Verify Activation
   - Upon success, you'll see the main dashboard
   - Check Settings > License to view license details

GRACE PERIOD: If your license expires, you have 7 days to renew before
the application stops working.

================================================================================
GETTING STARTED
================================================================================

After activation, Qanuni is ready to use:

1. FIRM SETUP
   - Go to Settings > Firm to enter your firm details
   - Add lawyers in the Lawyers module
   - Configure billing rates and expense categories

2. CLIENT & MATTER MANAGEMENT
   - Add clients in the Clients module
   - Create matters (cases) for each client
   - Track deadlines, hearings, and tasks

3. TIME TRACKING & BILLING
   - Log time entries for billable work
   - Record expenses and advances
   - Generate invoices automatically

4. COURT MANAGEMENT
   - Use the Calendar to view hearings and appointments
   - Set reminders for important dates
   - Track judgments and case outcomes

5. CORPORATE SECRETARY
   - Manage companies, shareholders, directors
   - Track corporate actions and resolutions
   - Generate corporate reports

================================================================================
DATA LOCATION
================================================================================

Your data is stored locally on your computer:

DATABASE: %APPDATA%\Qanuni\qanuni-web.db
LOGS:     %APPDATA%\Qanuni\logs\

(%APPDATA% typically resolves to C:\Users\[YourUsername]\AppData\Roaming)

IMPORTANT: Backup your database file regularly to prevent data loss.

================================================================================
UNINSTALLATION
================================================================================

To uninstall Qanuni:

1. Go to Windows Settings > Apps > Installed apps
2. Find "Qanuni" in the list
3. Click "Uninstall"

NOTE: Your database file will NOT be deleted automatically. If you want to
remove all data, manually delete the folder: %APPDATA%\Qanuni
